Movie list
  • React JS
  • Material UI

Movie list is an app that helps users to get informed about the latest movies and tv shows, look for a specific title or filter through hundreds of thousands of titles and save them for later.

The idea behind this project was to simplify the task of finding a movie or a TV show, by creating an easy-to-use UI and navigation.

The layout is split in 2 parts:

  • Navigation section
  • Main section
The navigation section contains a search input and navigation. Using react routes I created 5 different routes, making searching within reach and the userâ€™s journey on the app easy and very clear on his location.

Main section is where the information is shown. Titles are presented in card components informing the user about the score, title, genre and has the titleâ€™s poster as a background. Trending and Newest routs, displays 20 titles by default with the possibility to fetch for more, once the show more button is pressed. Upcoming and Discover, uses infinite scroll to fetch for more data as the use scrolls. In Watch list, you can find the titles that you have saved. This was achieved by using react context.

Once a title is selected, you will be taken to its page where you will find information like title, genre, language, release date, runtime and description of a movie. For tv shows you will get additional information, like status if the series is still airing, airing date when the series started and ended or when the next episode will air, seasons how many seasons it has, total episodes how many episodes.

In bough movies and tv shoes, you can see a trailer video, check its cast and get some recommended titles.

Weather app
  • React JS

Weather app helps users search a location or by using the browser location, get information about weather in their city and cities near them.

To build this application, I used React js and CSS. It uses SWR hook for fetching data and displays them in separated components like current weather, by week and by hour.

The current weather component is split in two, the local time and weather information like degrees, a description, feels like temperature and wind speed.
The week weather is presented as cards showing the day of the week, the date, description, day and night temperatures.

Hourly forecast is presented by a graph which shows the minimum and maximum temperatures in the span of 24 hours.
